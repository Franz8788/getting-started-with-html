Introduction to coding

The internet in general

The internet is the largest global computer network. How does it work? Basically, we need the following elements for it to run: Servers, the internet, the HTTP protocol (Hypertext transfer protocol), and a browser to read the files that are transferred.

HTML and CSS

HTML is the language used on the internet. The abbreviation stands for Hypertext Markup Language meaning that here is not only text, but also rules how the text is structured. CSS (Cascading Style Sheets) concern the presentation of a document, which can be changed by changing the CSS.

HTML elements

HTML elements consist of contents and tags. Tags are made up of tag names and angled brackets surrounding the tag names. Often the contents are wrapped by opening tags and closing tags. But there are also void tags such as <br> (Break). <br> stands alone and needs no closing tag later in the text. Other examples for tags are: <b> (bold, opening tag), </b> (bold, closing tag), <em> and </em> for italicized parts, <p> and </p> for opening and closing a paragraph.

Inline and block elements

In general, inline elements are flowing with the text of the document (for example: <img>), block elements have an invisible box around them (for example: <p> and < /p>). But we were assured that for beginners it is not necessary to understand the differences in every detail.

Links and images

Links are inline elements. Same goes for images. An example for an inactive link is (included here for clarification): <a href=”www.udacity.com”>inactive content</a>. An example for an active link is: active content. href represents an attribute. In the case of images the attribute is not href, but src.

Attention: Do not mix up src and href.
